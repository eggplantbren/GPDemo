A basic GP demo in DNest4
=========================

(c) 2018 Brendon J. Brewer

LICENSE: MIT.

To obtain and compile (requires Eigen3):

```
git clone --recursive https://github.com/eggplantbren/GPDemo
cd GPDemo
make
```

To run:

```
./main -t 8
```

When done, run

```
python showresults.py
```
